My portfolio is now on its second iteration. The first was made before I entered school, and it has been in need of a new face for quite some time now. I built this iteration of my webpage on top of static html, css, jQuery, javascript, and a framework called Foundation. The main animation was made with Adobe Animate CC (basically a repackaged Flash once Flash became a bad word), and converted to web goodness with a Javascript library called createJS that does all the heavy lifting.

Version 3 will not look any different then the current version, but I will be converting the whole website over to a PHP framework connected to a MySQL database with a custom lightweight CMS made to keep this site more up to date as I keep working towards become the best Web Developer and Designer I can.
